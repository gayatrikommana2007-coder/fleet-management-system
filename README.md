# Fleet Management System

## Project Overview
This project is a Fleet Management System developed using Python and Object-Oriented Programming (OOP). It simulates the management of multiple robots by allowing robot registration, task assignment, status monitoring, and task completion tracking.

## Features
- Robot Registration
- Task Assignment
- Fleet Status Monitoring
- Task Completion Tracking
- Multi-Robot Coordination

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
